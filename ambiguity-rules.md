# Ambiguity Rules

Storage technology terms often look like ordinary English. Resolve by context before translating.

| English | Preferred handling | Notes |
|---|---|---|
| box | 潜影盒 / 盒子 | In storage-tech context this usually means shulker box. |
| loader | 打包机 / loader | Usually fills shulker boxes with items. |
| unloader | 拆包机 / 卸货机 | Removes items from boxes. |
| splitter | 分盒器 / splitter | In box storage, often splits mixed boxes into per-item boxes. |
| merger | 合并器 / merger | Usually merges boxes or item streams. |
| slice | 单片 / slice | Repeated module of a storage system; often keep `slice`. |
| bulk | 大宗 | Not "large block". |
| bulk box | 大宗盒 / bulk box | Usually a box containing one bulk item type. |
| mixed box | 混杂盒 | A box containing multiple item types. |
| loose item | 散装物品 | Item not packed in a box. |
| blocker | 占位物 / 填充物 | Item used to occupy slots or constrain flow. |
| filter item | 过滤物品 | Item used to set or restore filter state. |
| cart | 矿车 / 漏斗矿车 | In storage-tech text, often hopper minecart. |
| item pile | 物品堆 / item pile | Clustered item entities used for merging, batching, or allocation. |
| temp | 暂存 / temp | Often temporary storage system, not temperature. |
| cache/buffer | 缓存 / 缓冲 | Distinguish only if the document does. |
| reset | 重置 / 复位 | Choose by mechanism. |
| call a slice | 调用某个 slice | Usually route work to a slice. |
| whitelist | 白名单 / 白名单过滤 | Storage-tech filter set, not security permission. |

Preserve English for unclear abbreviations such as `cycvar`, `svar`, `MIS`, or project-specific names unless the local glossary defines them.
